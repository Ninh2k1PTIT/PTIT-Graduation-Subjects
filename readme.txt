1. Cai dat MySQL
	- MySQL: https://dev.mysql.com/downloads/mysql/
	- MySQL workbench: https://dev.mysql.com/downloads/workbench/
2. Cai dat Front-end:
	- Cai dat Node.js: https://nodejs.org/en/download/current
	- Mo folder front-end bang cmd:
		+ Nhap 'npm i' de cai dat cac thu vien
		+ Nhap 'npm start' de chay ung dung
	- Ung dung chay tren http://localhost:4200
3. Cai dat Back-end
	- Cai dat JDK 8: https://www.oracle.com/java/technologies/javase/javase8-archive-downloads.html
	- Cai dat Maven: https://maven.apache.org/install.html
	- Mo folder back-end bang cmd: go 'mvn spring-boot:run'
	- Ung dung chay tren http://localhost:8080
